LAVORO DI TEAM - Realizzare un NUOVO progetto Laravel a tema Blog. Le features richieste sono le seguenti:

    Una Homepage accattivante
    Una pagina di visualizzazione degli articoli del Blog
    Una pagina di creazione di un articolo
    Una pagina chiamata Lavora con Noi

Queste features non devono essere tutte accessibili, soltanto Homepage e pagina di visualizzazione articoli sono accessibili ad utenti non autenticati. Per poter visualizzare anche le altre pagine bisogna prima registrarsi al portale. 

Features specifiche per pagina:

    Visualizzazione articoli: una lista di tutti gli articoli pubblicati con una card (per ogni articolo) che mostri un'immagine ed il titolo dell'articolo. Nella card deve essere presente un link per accedere al dettaglio
    Creazione articolo: un form con diversi campi di input tra cui titolo, autore dell'articolo, immagine, descrizione e poi dei campi check con le categorie. Le categorie ammesse per ogni articolo sono:
    Sport
    Cronaca
    Musica
    Arte
    Pubblicità
    Pagina di dettaglio: un header personalizzato dove siano visibili tutti i dati dell'articolo, compresa l'immagine che è stata caricata ed una lista di tutte le categorie a cui appartiene.
    Lavora con noi: un form in cui inserire i propri dati, come nome, cognome, mail e content. Al submit del form deve partire una mail che arriverà a mailtrap con il messaggio scritto (ricordate di non usare la keyword message per le mail perchè non funziona)

LAVORATE IN PAIR-PROGRAMMING: scrive un solo buddy per volta e ci si alterna ogni 30 minuti. Ogni volta che un membro del team finisce deve pushare tutto sulla repository e gli altri buddy pullano. Ricordate, prima di tutto, di creare la repository e tutti clonano il progetto rendendolo funzionante e poi si comincia a lavorare.

SPUNTI FRONTEND: lavorate anche sul frontend della pagina, qui trovate un link dove potete trovare alcuni template da cui prendere spunto ma siete liber* di ispirarvi anche ad altri siti web.
